# AGROFARM - Sistema de Gestión Porcícola

Este proyecto es una demo educativa que simula una plataforma de administración para una granja porcícola. Utiliza **HTML5**, **TailwindCSS** y **JavaScript** para mostrar de forma sencilla las distintas áreas de un sistema real.

## Estructura del proyecto
- `index.html` – Página de inicio y acceso.
- `register.html` – Formulario de registro con selección de rol.
- `dashboard.html` – Panel principal con módulos y reportes.
- `/scripts` – Archivos JavaScript modulares (`auth.js`, `dashboard.js`, `pigs.js`, `charts.js`).

El menú muestra diferentes módulos según el rol (Administrador, Veterinario u Operario) y un pequeño breadcrumb indica la sección actual.

## Uso
1. Abre `index.html` en tu navegador.
2. Registra un usuario indicando su rol y luego inicia sesión.
3. Desde el dashboard podrás gestionar cerdos, registrar eventos y generar un pequeño reporte descargable.

Esta demostración no implementa un backend real; su propósito es mostrar la apariencia y estructura de un sistema de gestión porcícola.
